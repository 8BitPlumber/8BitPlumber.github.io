Livesplit Setup

Livesplit Tutorial(Windows Only - For other OS, use one.livesplit.org)

Livesplit is a popular timer program used by speedrunners to track their performance during runs. Follow these steps to set up Livesplit:

  1. Download Livesplit from the official website: livesplit.org.
    • Navigate to "Downloads" and download the latest version.
  2. Extract the folder and run the "Livesplit" application.
  3. Setting up the Layout:
    • When LiveSplit opens, you'll see a default layout with a timer.
    • Right-click on the timer and select "Edit Layout" to customize your layout. You can resize and move components by clicking and dragging them.
    • You can also add or remove components using the right-click menu.
  4. Adding Splits
    • To add splits, right-click on LiveSplit and select "Edit Splits."
    • Click on the "+" button to add a split.
    • Enter the name of the split (e.g., "World 1-1") and press Enter.
    • Repeat this process for each split in your speedrun.
  5. Configure Livesplit's hotkeys for starting, splitting, resetting, etc.
  6. Setting up Additional Components:
    • To add additional components, right-click on LiveSplit and select "Edit Layout."
    • Click on the "+" button to add a new component.
    • Choose the component you want to add from the list (e.g., Timer, Sum of Best, Current Pace, etc.).
    • Customize the settings for each component as desired.
  7. Using an Auto Splitter
    • If you want to use an auto splitter for Super Mario Bros 1, you'll need to download the appropriate script for LiveSplit.
    • Go to the auto splitter website at github.com/periwinkle9/smb-autosplitter.
    • Navigate to the "Releases" section on the right hand side and download the latest file.
    • Once downloaded, open LiveSplit and go to the "Edit Splits" menu.
    • Click on the "Activate" button next to the auto splitter option and select the script you downloaded.
    • Configure any additional settings required by the auto splitter, such as starting the timer automatically when the game starts and RNG manipulation.
  8. Load your game and start using Livesplit to time your runs!
