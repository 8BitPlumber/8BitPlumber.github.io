Emulator & TASing Setup

Part 1: How to Set up an Emulator for RTA

Introduction:

In this section, we'll cover the setup process for two main emulators used in Real-Time Attack (RTA) speedrunning of Super Mario Bros 1: Mesen RTA and Nestopia.

MesenRTA (Only Available On Windows)

  1. Download MesenRTA
    • Visit the MesenRTA website at github.com/threecreepio/mesenrta.
    • Click "Releases" in the right-hand column.
    • Download the Mesen.zip file.
  2. Extract the Mesen.zip file and run the application.
    • To play a game, go to File > Open and select your ROM.

Nestopia

  1. Download Nestopia
    • Visit the Nestopia website at nestopia.sourceforge.net.
    • Navigate to the "Downloads" section and download the correct version for your operating system.
  2. Extract the file and run the application.
    • To play a game, go to File > Open and select your ROM.

Part 2: How to Set up an Emulator for TASing

Introduction:

In this section, we'll cover the setup process for two main emulators used in Tool-Assisted Speedruns/Superplays (TAS) of Super Mario Bros 1: FCEUX and Bizhawk.

Bizhawk

  1. Download Bizhawk
    • Visit the Bizhawk website at github.com/TASEmulators/BizHawk.
    • Click "Releases" in the right-hand column.
    • Download the correct release for your operating system.
  2. Extract the file and run the EmuHawk application.
    • To play a game, go to File > Open ROM and select your ROM.
    • To create a TAS, go to Tools > TAStudio.

FCEUX

  1. Download FCEUX
    • Visit the FCEUX website at fceux.com.
    • Navigate to the "Download" section and download the most recent version for your operating system.
  2. Extract the file and run the FCEUX application.
    • To play a game, go to File > Open and select your ROM.
    • To create a TAS, go to Tools > TAS Editor.
